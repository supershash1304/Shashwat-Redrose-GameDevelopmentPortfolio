Highlights

  • This project will be the development of a 2D endless runner game that integrates large language model (LLM) technology to enable real-time, voice-controlled character interaction alongside a traditional, keyboard-controlled player. The project combines fast-paced gameplay with innovative human–AI collaboration, exploring how generative AI can enhance interactivity and immersion. Objectives Objectives are: 1. To develop a functional 2D endless runner with moderate obstacle density. 2. To integrate an LLM to interpret voice commands. 3. To enable real-time speech-to-text and text-to-speech interaction. 4. To build cooperative gameplay between AI and player characters. Secondary Objectives: - Test user engagement improvement from LLM-driven gameplay. - Demonstrate hybrid AI-human control in real-time systems. Project Description Gameplay involves a 2D side-scrolling endless runner where one player uses the keyboard while the second, AI-controlled character responds to voice commands. The AI interprets commands using an LLM, producing contextual dialogue and game actions.
